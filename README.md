# ironman_data-race-analysis
This repository contains an exploratory data analysis of Ironman race results using R. The project focuses on understanding athlete performance, finishing trends, and participation across countries through data cleaning and visualization.

What This Project Does:

Cleans and prepares Ironman race data

Analyzes race segment performance (swim, bike, run)

Examines total finish times and finisher status

= Visualizes country-wise distribution of finishers

Tools Used:

R

tidyverse

ggplot2

R Markdown

Output:

The final output is an HTML report generated using R Markdown that includes visualizations and brief interpretations.
